# fly-high

* Clone repository
* run `pip install`
* run `python app.py`

server will start on `http://127.0.0.1:5000`

Python version used `3.8.12`

API endpoint:

* `http://127.0.0.1:5000/plan` : POST request to search the trip details.
  * Request:
    ```
      {
        "source": "OBY"
      }
    ```
